# E11 习惯性未验算

## 定义
学生做完题目后不检查或不会验算，导致明显错误的答案没有被及时发现。

## 常见表现
- 结果与估算差距很大但没有发现
- 减法结果比被减数大
- 乘法结果比加法还小
- 应用题答案明显不符合常理

## 教学工具
- 估算法（先估算大概范围，再精确计算）
- 逆运算法（加法用减法验算，乘法用除法验算）
- 代入法（把答案代回原式检查）

## 引导步骤（标准模式）
1. 接纳：做完了不等于做对了，我们来检查一下。
2. 引导：你觉得答案大概应该是多少？你的答案和估算差得远吗？
3. 总结：做完题目花10秒钟检查，能减少很多不必要的错误。
4. 练习：做几道题，每道做完后用估算检查一遍。

## 年级对应
- 一至二年级：简单加减法逆运算检查
- 三至四年级：估算 + 逆运算
- 五至六年级：代入法 + 量纲检查
