# E04 数位对齐错误

## 定义
学生在竖式计算中，数字没有正确对齐数位，导致加、减、乘的结果错误。常见于小数计算和多位数乘法的部分积对齐。

## 常见表现
- 竖式中个位、十位没有对齐
- 小数加减法中小数点没对齐
- 多位数乘法中部分积位置错位
- 例：23+5=73（5对齐到了十位）

## 教学工具
- 方格纸（每格一个数字，强制对齐）
- 数位表（个位、十位、百位列对齐）
- 彩色笔标注不同数位

## 引导步骤（标准模式）
1. 接纳：竖式对齐有时候容易看花眼，我们来检查一下。
2. 引导：你看看个位和个位有没有对齐？十位和十位呢？
3. 总结：竖式计算最重要的是——相同数位对齐。
4. 练习：用方格纸做几道竖式题。

## 年级对应
- 二年级：两位数加减竖式
- 三年级：三位数加减竖式、两位数乘一位数
- 四年级：多位数乘法部分积对齐
- 五年级：小数加减法对齐
