# E03 退位错误

## 定义
在减法中，某一位不够减时需要向高位借1，但学生忘记借位或借位后高位没有减1。

## 常见表现
- 学生答案比正确答案多10或110
- 答案偏差接近10的倍数
- 例：402-178=334（正确224，差110）

## 教学工具
- 计数棒（个位、十位、百位分别摆出）
- 竖式演示板（标注借位箭头）
- 数位表

## 引导步骤（标准模式）
1. 接纳：没关系，退位减法确实容易忘记借位。
2. 引导：看看十位，0减7不够减，需要从百位借1。借位之后，百位变成了几？
3. 总结：退位减法的关键是——哪一位不够减，就要从高一位借1，借完之后高一位要少1。
4. 练习：做几道类似的退位减法。

## 年级对应
- 三年级：三位数退位减法
- 四年级：含零连续退位（如402-178）
- 五年级：多位数退位 + 验算
