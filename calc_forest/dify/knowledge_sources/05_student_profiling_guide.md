# 学生画像分析引导手册

## 画像理念

- 画像是帮助教师理解学生的工具，不是给学生贴标签。
- 所有画像描述使用积极、温暖、具体的语言。
- 画像关注可改进行动，不做能力定性判断。
- 画像数据来源于历史做题统计，不引入主观评价。

---

## 学生类型描述

### 扎实型
- 特征：准确率稳定在80%以上，各错因类型的表现均衡
- 描述语言："计算基础扎实"、"各维度表现均衡"
- 建议：可以适当增加挑战题，探索更多解题方法

### 稳步型
- 特征：准确率在60%-80%之间，有1-2个薄弱环节但整体趋势向上
- 描述语言："稳步前行"、"进步空间明确"
- 建议：集中练习薄弱环节，保持每日短时训练

### 努力型
- 特征：准确率在40%-60%之间，做题量大但错误也多
- 描述语言："正在努力"、"练习认真"
- 建议：降低难度，先巩固基础再逐步提升

### 成长型
- 特征：准确率在60%以下但近期有明显进步趋势
- 描述语言："进步明显"、"正在追赶"
- 建议：肯定进步，维持当前练习节奏

## 错因模式解读

### 单一错因集中型
- 特征：错误集中在某一种错因代码（如全是E03）
- 解读：该知识点存在理解偏差，但其他方面正常
- 建议：针对性地讲解该知识点，不需要全面补课

### 多种错因分散型
- 特征：错误分散在多种错因代码
- 解读：可能是计算习惯问题，而非某个知识点没掌握
- 建议：关注做题过程，培养检查习惯

### 基础事实密集型（E01高频）
- 特征：基础事实错误占比最高
- 解读：口诀或基础运算不熟练
- 建议：每日5分钟口算训练，不加大题量

### 运算顺序反复型（E05高频）
- 特征：运算顺序错误反复出现
- 解读：对运算优先级的理解不够稳固
- 建议：用"圈出先算部分"的方法练习

## 学习风格识别线索

### 从错误模式推断
- 步骤遗漏多（E08）→ 可能是跳跃式思维，需要引导写完整步骤
- 抄题错误多（E07）→ 可能是注意力分配问题，需要训练审题习惯
- 未验算多（E11）→ 可能是缺乏自我检查意识，需要建立验算习惯
- 算理错误多（E09）→ 可能是需要更直观的理解方式，建议用图示教学

### 从准确率趋势推断
- 准确率稳步上升 → 适合当前教学节奏
- 准确率波动较大 → 可能受情绪或注意力影响
- 准确率持续下降 → 需要教师关注，了解是否有外部因素

## 成长叙事写作指南

### 核心原则
- 使用温暖鼓励的语言
- 关注进步而非不足
- 给出具体可操作的下一步建议
- 不做排名或比较

### 叙事模板

#### 高准确率学生
"[学生]这阶段的准确率达到了[XX]%，计算能力扎实。[如果有薄弱点]不过在[错因代码]类型的题目上还有提升空间，建议多练几道同类题。继续保持，小树苗在茁壮成长！"

#### 中等准确率学生
"[学生]准确率[XX]%，正在稳步前进。重点关注[错因代码]类型的题目，多练几次就会更熟练。每一步努力都在让小树长高一点点。"

#### 需要关注的学生
"[学生]准确率[XX]%，没关系，我们一起一步一步来。先从[最需要关注的错因]类型的题目入手，找到规律就好了。每棵小树都有自己的成长节奏。"

### 不建议使用的表达
- "该学生基础较差" → 改为 "基础运算还有巩固空间"
- "成绩退步" → 改为 "本阶段的练习准确率有波动"
- "需要多做题" → 改为 "建议每天做5分钟针对性练习"
- "跟不上进度" → 改为 "当前节奏可以再慢一点，先确保基础稳固"

## 个性标签词汇表

### 积极标签（推荐使用）
- 计算扎实、稳步前行、进步明显、各维度均衡
- 正在努力、练习认真、敢于尝试、善于思考
- 方法灵活、步骤清晰、习惯良好

### 温和描述标签
- 基础需巩固、进退位需关注、运算顺序需关注
- 审题习惯待培养、验算意识待加强
- 正在追赶、进步空间明确

### 禁止使用的标签
- 差、弱、落后、不行、太慢
- 不认真、不努力、不专心
- 倒数、最差、拖后腿
