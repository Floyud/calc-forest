# 运算律 — 乘法分配律

## 计算法则
1. 乘法分配律：(a+b)×c = a×c + b×c
2. 逆用：a×c + b×c = (a+b)×c（提取公因数）
3. 拓展：(a-b)×c = a×c - b×c
4. 凑整技巧：看到能凑成整十整百的数，优先用分配律

## 典型例题
- 25×(40+4) = 25×40 + 25×4 = 1000+100 = 1100
- 99×35 = (100-1)×35 = 3500-35 = 3465
- 38×7 + 62×7 = (38+62)×7 = 100×7 = 700
- 102×43 = (100+2)×43 = 4300+86 = 4386

## 常见错误
- E-K07 | 只分配了一项 | 25×(40+4) = 25×40+4 = 1004 | 两项都要乘：25×40+25×4
- E-K15 | 加法分配律和乘法结合律混淆 | (25×4)×8 用了分配律 | 括号里是"+"才用分配律，是"×"用结合律
- E-K07 | 逆用时漏项 | 38×7+62×7 = 38+62×7 | 提取公因数后括号里是(38+62)

## 检测规则
- 正用验证：展开后各项之和 = 原式
- 逆用验证：提取公因数后，括号内各项之和 × 公因数 = 原式
- 若展开后项数少了，判定E05（漏分配）

## 教学提示
- 用面积模型解释：长(a+b)×宽c = 长a×宽c + 长b×宽c
- 分配律是简便计算的核心工具，反复练习正用和逆用
