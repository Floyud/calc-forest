# 圆 — 圆的周长和面积

## 计算法则
1. 圆的周长 C = πd = 2πr（d=直径，r=半径）
2. 圆的面积 S = πr²
3. π ≈ 3.14（小学阶段统一取3.14）
4. 半圆周长 = πr + 2r（半圆弧+直径）
5. 圆环面积 = π(R² - r²)

## 典型例题
- r=5cm → C=2×3.14×5=31.4cm, S=3.14×25=78.5cm²
- d=8cm → r=4cm → C=3.14×8=25.12cm, S=3.14×16=50.24cm²
- 半圆r=3cm → 周长=3.14×3+6=15.42cm

## 常见错误
- E-K10 | 周长和面积公式混 | 面积=2πr（用了周长公式） | 周长=2πr，面积=πr²
- E-K02 | π口算错 | 3.14×5=17.5（应为15.7） | π×r的口算要熟练
- E-H11 | r²算错 | r=5, r²=10（应为25） | r²=r×r，不是r×2
- E-K10 | 半圆周长忘加直径 | 半圆周长=πr（应为πr+2r） | 半圆周长=半圆弧+直径

## 检测规则
- 周长验证：C = 2×π×r 或 C = π×d
- 面积验证：S = π×r²
- r² = r × r（不是r×2）
- 半圆周长 = πr + 2r（不是πr）

## 教学提示
- π口算表必须背熟：3.14×2=6.28, ×3=9.42, ×4=12.56, ×5=15.7...
- r²是面积公式的关键，很多学生算错
