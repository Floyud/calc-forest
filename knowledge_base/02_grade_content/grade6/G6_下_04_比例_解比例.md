# 比例 — 解比例

## 计算法则
1. 比例：两个比相等的式子 a:b = c:d
2. 比例的基本性质：内项积=外项积（ad=bc）
3. 解比例：用交叉相乘求未知项
4. 若 a:b = c:x → x = bc/a

## 典型例题
- 2:3 = 8:x → 2x=3×8=24 → x=12
- x:5 = 6:10 → 10x=5×6=30 → x=3
- 1.5:0.5 = x:2 → 0.5x=1.5×2=3 → x=6

## 常见错误
- E-K03 | 交叉相乘口诀错 | 2:3=8:x → 2x=3+8=11 | 交叉是乘不是加
- E-H07 | 交叉相乘多位数进位错 | 3×8=24对，但12×15算错 | 交叉相乘中的乘法要准确
- E-H08 | 内项外项搞混 | a:b=c:d → ab=cd（应为ad=bc） | 交叉=外项×外项=内项×内项

## 检测规则
- 内项积 = 外项积
- 解比例：未知项 = 对角两项之积 ÷ 已知项
- 验证：把x代入原比例，两个比的比值相等

## 教学提示
- "交叉相乘"画箭头帮助学生理解
- 和分数等式对比：2/3=8/x 就是 2x=24
