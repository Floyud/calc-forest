# 六年级上册计算法则速查表

本文件为六年级上册所有计算法则的快速参考，每条法则含公式、示例和常见错因代码，供 Dify 知识库快速检索。错因代码详细定义参见04_error_taxonomy。

## 分数乘法

**分数×整数：** 分子×整数作分子，分母不变，能约分先约分。
公式：a/b × n = (a×n)/b
例：2/5×3 = 6/5，3/4×8 = 3×2 = 6（先约分：4和8约）
错因：E01（口诀错）、E06（误把分母也乘整数）

**分数×分数：** 分子×分子，分母×分母，先交叉约分再计算。
公式：a/b × c/d = (a×c)/(b×d)
例：2/3×3/4 = 1/2，5/6×3/10 = 1/4，7/12×8/21 = 2/9
错因：E01（约分后乘法错）、E06（忘记约分）、E09（不理解交叉约分原理）

**分数×小数（先约分）：** 小数能与分母直接约分时优先使用，最简便。
例：2.4×3/4 → 2.4÷4=0.6 → 0.6×3=1.8
错因：E09（不会判断能否约分）

**分数×小数（化小数）：** 分母只含2和5因数时，分数化有限小数相乘。
例：1/4×0.3 = 0.25×0.3 = 0.075
错因：E06（小数位不够忘补0）

**分数×小数（化分数）：** 小数化分数后按分数乘法算，所有情况都适用。
例：2/3×0.6 = 2/3×3/5 = 2/5
错因：E06（小数化分数出错）

**方法选择口诀：** 一看能否先约分，二选方法看数据，三算结果要仔细。

**乘法交换律：** a×b = b×a。例：2/5×1/3 = 1/3×2/5 = 2/15。错因：E10（与结合律混淆）

**乘法结合律：** (a×b)×c = a×(b×c)。例：(2/3×3/4)×4/5 = 2/3×(3/4×4/5) = 2/5。错因：E10

**乘法分配律：** (a+b)×c = a×c+b×c。反向：提取公因数。
凑整技巧：5/6×19+5/6 = 5/6×(19+1) = 5/6×20 = 50/3
错因：E05（展开漏项）、E09（不会反向提取）

## 分数除法

**倒数：** 乘积为1的两个数互为倒数。求倒数=交换分子分母位置。
特殊规则：1的倒数是1，0没有倒数。带分数先化假分数再颠倒。
例：3/5的倒数是5/3，7的倒数是1/7
错因：E01（带分数未化假分数）、E06（整数倒数写错）

**分数÷整数：** 除以整数（0除外）=乘以该整数的倒数（变除为乘）。
公式：a/b÷n = a/b×1/n
例：4/5÷2 = 4/5×1/2 = 2/5，3/4÷3 = 1/4
错因：E01（忘变除为乘）、E06（误把分母也除以整数）

**分数÷分数：** 除以一个分数=乘以它的倒数（变除为乘）。
公式：a/b÷c/d = a/b×d/c
例：2/3÷4/5 = 2/3×5/4 = 5/6，5/6÷5/8 = 4/3，6÷2/3 = 9
错因：E01（颠倒了被除数而非除数）、E09（不理解为什么要颠倒）

**分数混合运算：** 先乘除后加减，有括号先算括号内，同级从左到右。
例：1/2+1/3×3/4 = 1/2+1/4 = 3/4（先乘后加）
例：3/4÷(1/2+1/4) = 3/4÷3/4 = 1（先算括号）
错因：E05（运算顺序错）、E08（跳步遗漏）、E06（加减未通分）

## 比

**求比值：** 前项÷后项，结果是一个数（整数、小数或分数）。
例：3:4的比值 = 3÷4 = 0.75（或3/4）。注意：比值不带冒号。
错因：E04（与化简比混淆）、E06（除法计算出错）

**化简整数比：** 前后项同时除以最大公因数。
例：12:18 = (12÷6):(18÷6) = 2:3。结果前后项互质。
错因：E01（GCD算错）、E04（未化到最简）

**化简分数比：** 同乘分母最小公倍数化为整数比，再按整数比化简。
例：1/3:1/4 = (×12) = 4:3
错因：E06（LCM算错）、E04（未化最简）

**化简小数比：** 同乘10的幂化为整数比，再化简。
例：0.6:0.9 = (×10) = 6:9 = 2:3
错因：E06（小数位移出错）

**按比分配：** 总份数=各项之和，总量÷总份数=每份量，每份量×份数=对应量。
例：120按2:3分配 → 每份=120÷5=24 → 48和72
错因：E10（理解错比的含义）、E08（忘记先算总份数）

## 百分数

**小数化百分数：** 小数点右移两位加%。例：0.35=35%，0.125=12.5%。错因：E06（方向搞反）

**百分数化小数：** 去%左移两位。例：68%=0.68，125%=1.25。错因：E06（方向搞反）、E04（忘补0）

**分数化百分数：** 先化小数（分子÷分母），再化百分数。
例：3/4=0.75=75%，3/8=0.375=37.5%
错因：E01（除法出错）、E06（移位方向错）

**分数化小数：** 分子÷分母。分母只含2和5因数时可化有限小数。
例：3/8=0.375，2/5=0.4。错因：E01

**小数化分数：** 用10的幂做分母再约分。
例：0.6=6/10=3/5，0.375=375/1000=3/8。错因：E06（约分不彻底）

**打折计算：** 原价×折扣%=售价。"八折"=80%=0.8。
例：200元八折=200×0.8=160元。错因：E10（折扣概念混淆）、E09

**求百分之几：** 部分÷整体×100%。例：25÷200=12.5%。错因：E10（分子分母搞反）

**纳税与利率：** 本金×利率×时间=利息。
例：1000元，年利率3%，存2年：1000×3%×2=60元。错因：E10、E09

## 圆

**圆的周长：** C=2πr或C=πd，π取3.14。
例：r=5cm→C=2×3.14×5=31.4cm；d=10cm→C=3.14×10=31.4cm
错因：E01（3.14×2算错）、E02（进位遗漏）

**圆的面积：** S=πr²，先算r²再乘3.14。
例：r=4cm→S=3.14×4²=3.14×16=50.24cm²
错因：E01（r²算成r×2）、E04（小数点错位）

**环形面积：** S=π(R²-r²)，大圆面积减小圆面积。
例：R=5cm,r=3cm→S=3.14×(25-9)=3.14×16=50.24cm²
错因：E08（忘记减内圆）、E05（运算顺序错）
