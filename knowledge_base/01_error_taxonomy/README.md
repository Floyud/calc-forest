# 错因分类体系 v2.0

分为两大类：
- **knowledge/** — 知识相关错因（E-K）：绑定具体知识点，换一个知识点就不会犯
- **habit/** — 习惯相关错因（E-H）：跨年级共性坏习惯，3-6年级不论知识都可能犯

---
