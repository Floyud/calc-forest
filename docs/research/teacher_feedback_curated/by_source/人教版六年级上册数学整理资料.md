一、分数乘法（第一单元）
这是分数计算的起点，核心是理解"求一个数的几分之几"用乘法。
·分数乘整数：分子乘整数作分子，分母不变；能约分的先约分。
例：×3=
·分数乘分数：分子相乘的积作分子，分母相乘的积作分母。
例：x===
·分数乘小数：可将小数化成分数，或将分数化成小数来计算。如果小数能与分母约分，直接约分更简
便。
例：2.5×I可以写成×=，或2.5×0.6=1.5
·运算律推广：整数乘法的交换律、结合律、分配律在分数乘法中同样适用，用于简便运算。
二、分数除法（第三单元）
分数除法是分数乘法的逆运算，关键在于"变除为乘”。
·倒数：乘积为1的两个数互为倒数。
。求一个数的倒数：交换分子分母的位置。整数可以看作分母为的分数。
。1的倒数是1，0没有倒数。
·分数除以整数：等于分数乘以这个整数的倒数。
例：÷2=×=
一个数除以分数：等于这个数乘以除数的倒数。
例：×=
·混合运算：先乘除，后加减，有括号先算括号里的。除以一个数（0除外）等于乘以它的倒数，可将除法
混合运算统一转化为乘法再计算。
```

三、比(第四单元）
比与除法、分数关系密切，是本学期"算术"向"代数"过渡的桥梁。
·求比值：用比的前项除以后项，结果可以是整数、小数或分数。
例：6:8=6÷8=0.75或
·化简比：利用比的基本性质（前项和后项同时乘或除以同一个不为0的数，比值不变）。
。整数比：除以最大公因数。
。分数比：乘分母的最小公倍数。
。小数比：先化成整数比，再化简。
。最终结果必须写成a：b的形式。
四、百分数（第六单元）
百分数实质是分母为100的分数，但在计算和形式上与分数有区别。
·百分数与小数互化：
。小数→百分数：小数点向右移动两位，加上百分号。例：0.75=75%
。百分数→小数：去掉百分号，小数点向左移动两位。例：45%=0.45
·百分数与分数互化：
。分数→百分数：通常先化成小数（除不尽保留三位小数），再化成百分数。或化成分母为100的分
数。
·百分数应用中的计算（如求一个数的百分之几）：将百分数化为小数或分数后，再与该数相乘。
```
