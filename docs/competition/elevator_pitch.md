# 30 秒电梯演讲

> 用于比赛现场、评委提问、社交媒体简介等需要极短介绍的场合。
> 目标：30 秒内让听众理解"这是什么、为什么不一样、为什么值得关注"。
> 语速参考：约 4 字/秒，30 秒 ≈ 120 字。

---

## 中文版（主版本）

```text
我的计算森林，是一个帮小学数学老师"看见错因"的教育智能体。

传统工具只能告诉你"对"还是"错"。我们做了 E01 到 E11，一共 11 种纳米级错因分类——
这是目前国内唯一做到这个精度的小学数学诊断体系。

老师输入一道题和学生答案，两秒内得到：错因是什么、证据是什么、接下来练什么。
然后系统会用语音一步步引导孩子自己想明白——永远不直接给答案。
所有 AI 输出默认待教师审核，天然符合 2025 年未成年人 AI 使用政策。

每个孩子种一棵树，每做对一道题，树就长高一点。
没有排名，没有焦虑——只有一片森林，记录每个孩子的成长。
```

## 更短版本（15 秒）

## 英文版（备用）

```text
"Calculation Forest" is an AI-powered diagnostic agent for primary-school math.

Existing tools only say "right" or "wrong." We classify errors into 11 nano-level 
categories — the only system in China that tells you not just THAT a child made a 
mistake, but WHY: was it a borrow error, a carry error, or a place-value misalignment?

Teachers input a problem and a student answer. In two seconds, they get: the error 
type, the evidence, and what to practice next. The system then guides the child 
step-by-step using voice — never giving away the answer.

Every AI output is gated behind teacher review — natively compliant with China's 
2025 policy on AI use for minors under 13.

Each child grows a tree. No rankings, no anxiety — just a forest of growth.
```

## 核心记忆锚点（评委可能记住的关键词）

| 锚点 | 含义 |
|------|------|
| **E01-E11** | 唯一的纳米级错因分类体系（竞品只有对/错） |
| **教师审核门控** | 天然合规 2025 未成年人 AI 政策 |
| **引导不给答案** | 教育理念差异化 |
| **一片森林** | 情感记忆锚点（Duolingo 的猫头鹰 → 我们的计算森林） |
