# 演示视频脚本 v2

> 本文档替代 `demo_video_script.md`。
> 核心转变：从"展示 Dify 工作流节点 + JSON 输出"转向"以前端界面为主角、AI 为幕后的视觉叙事"。
> 目标观众：《创AI》评委 — 教育工作者 + AI 专家。
> 视频时长：约 3 分钟（180 秒）。

---

## 视频定位

**一句话**：让评委在 3 分钟内看见一个完整的、有温度的、超越"对错二分法"的小学数学教育智能体。

**视觉策略**：全程录制前端界面（calc_forest/web），AI 能力通过界面效果自然呈现，不暴露引擎内部。

## 录屏操作清单

按以下顺序操作，确保画面流畅：

1. **[第一幕]** 预制幻灯片/动画：黑屏 → 题目 → 红叉 → 多张错题闪过 → 定格 → 渐暗
2. **[第二幕]** 打开诊断台页面 `/diagnose`，填入样例，点击诊断，等待结果滑入。**慢速滚动**展示诊断卡片 → 教师摘要 → 练习推荐 → 回到"待教师审核"标记
3. **[第三幕]** 点击导航"引导预览"，展示 4 步进度条和第一步引导语。**切到**聊天页面 `/chat`，展示 2-3 轮对话（预输入或回放）。点击"朗读"按钮展示 TTS 动画。**切回**引导页展示全部 4 步完成
4. **[第四幕]** 点击导航"班级林园" `/forest`，**等待树木动画全部加载完成**（约 1-2 秒）。缓慢悬停不同树木展示 tooltip 和粒子效果。点击一棵树打开详情抽屉，依次切换 3 个标签页。**切到**课堂模式 `/classroom`，展示错因画像和生成流程。**短暂展示**全屏白板
5. **[第五幕]** 回到森林页，缓慢拉远/缩小浏览器窗口让森林全景可见。**使用预制动画**展示 8 周前后对比。展示 4 行价值主张。展示品牌 logo

## 拍摄注意事项

1. **全程录前端界面**，不出现 Dify 工作流编辑器、JSON 原始输出、终端命令行或代码编辑器。
2. 鼠标移动要**缓慢且有目的性**——每一步操作前先悬停 1 秒，让画面"呼吸"。
3. 所有动画（树木加载、结果滑入、粒子效果）**必须等其播放完毕**再进行下一步操作。
4. 如视频中出现 AI 生成的引导文本、语音，需按指南在画面角落标注 **"AI 生成"**。
5. 全程使用脱敏样例数据，不出现真实学生信息。
6. 旁白使用温和但专业的语气，语速约 4 字/秒（3 分钟约 720 字，当前旁白约 680 字，留有余量）。
7. 背景音乐建议使用轻柔的钢琴/弦乐，音量不超过旁白的 15%。
