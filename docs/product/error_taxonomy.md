# 小学数学计算错因标签体系

## 设计原则

错因标签要服务教师教学，不追求医学式诊断，也不把学生简单贴上“粗心”标签。每个标签都需要能回答三个问题：

- 这类错误表现是什么。
- 教师如何识别。
- 后续如何干预。

## 首版标签

| 标签 ID | 错因标签 | 典型表现 | 诊断证据 | 干预建议 |
| --- | --- | --- | --- | --- |
| E01 | 基础事实错误 | 口诀、口算、加减事实错误 | 单步计算结果与标准事实不一致 | 低负荷口算练习，强化事实提取 |
| E02 | 进位错误 | 加法应进位未进位，或进位数字加错 | 个位/十位局部结果缺少进位影响 | 标注进位位置，进行同位练习 |
| E03 | 退位错误 | 减法应退位未退位，或退位后被减数处理错误 | 某一位出现不够减但结果未体现退位 | 使用数位表和拆分法复盘 |
| E04 | 数位对齐错误 | 竖式位值错位，小数位未对齐 | 学生结果接近错位计算结果 | 强化位值意识和书写规范 |
| E05 | 运算顺序错误 | 先后顺序错误，括号处理错误 | 学生答案符合错误顺序计算 | 训练标注运算顺序和分步书写 |
| E06 | 小数点/分数单位错误 | 小数点位置错，分数单位混淆 | 数字主体接近但位值或单位错误 | 结合数轴、面积模型理解单位 |
| E07 | 抄题/转写错误 | 数字、符号、单位从题目到过程变化 | 学生步骤中的题目与原题不一致 | 建立读题、圈画、复核流程 |
| E08 | 步骤遗漏 | 中间步骤跳过导致结果错误 | 关键计算环节缺失 | 使用“每步一行”模板训练 |
| E09 | 算理理解不足 | 新情境或变式题中套错算法 | 错误不只发生在单步计算，而是方法选择错误 | 用图示、操作和反例讲清算理 |
| E10 | 审题与单位理解错误 | 问什么答什么不一致，单位换算错 | 计算本身可能正确但答案对象错误 | 训练关键词、单位和问题复述 |
| E11 | 未验算 | 明显不合理结果未被发现 | 结果超出估算范围或与常识矛盾 | 引入估算、逆运算、代入检查 |

## 标签粒度

首版保持 10-12 个一级错因，避免过细导致教师难以使用。后续可以在标签下增加二级细分，例如：

- `E02.1` 漏进位。
- `E02.2` 进位后忘记加。
- `E03.1` 未退位。
- `E03.2` 退位后本位未减一。

## 首版演示适用范围

首版演示聚焦整数计算，不把完整小学数学知识范围一次性做完。演示题型包括：

- 三位数加减法。
- 含 0 的连续退位减法。
- 一位数乘两位数。
- 简单四则混合运算。

首版规则和样例数据优先覆盖 `E01`、`E02`、`E03`、`E04`、`E05`、`E07`、`E08`、`E11`。其中：

- `E01` 覆盖加减口算事实和乘法口诀错误。
- `E02` 覆盖三位数加法和一位数乘两位数中的进位遗漏或进位后未加。
- `E03` 覆盖普通退位和含 0 连续退位。
- `E05` 覆盖简单四则混合运算中的先后顺序错误。

`E06` 和 `E10` 保留在完整 taxonomy 中，但暂不作为首版演示必测标签。

## 与练习推荐的关系

每个错因标签都要绑定练习策略：

- 基础事实错误：短时高频、低难度、即时反馈。
- 进位/退位错误：同一结构少量练习，加步骤提示。
- 数位对齐错误：书写格式练习和位值表。
- 运算顺序错误：先标顺序再计算。
- 算理理解不足：少题量、强解释、用图示和反例。

## 避免的标签

不使用“粗心”“笨”“态度差”等不可教学化标签。若表现为注意或习惯问题，应拆成可观察行为，例如“未验算”“抄题错误”“步骤遗漏”。
