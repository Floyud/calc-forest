Original teacher-authored feedback and design documents.

- `计算小树苗设计资料.docx`
- `第4次修改（5月2号）.docx`

How to use them:

- `docs/specs/teacher_feedback_digest.md`：提炼教师材料中的产品约束与 MVP 解释
- `docs/research/teaching_schedule_and_problem_design.md`：提炼教学进度、出题原则、教材整理资料对后续练习系统的影响
- `docs/research/teacher_feedback_curated/teacher_feedback_master_digest.md`：新版主题整理稿，整合 5 份教师材料并保留来源追踪
- `docs/research/teacher_feedback_curated/by_source/`：逐文件整理稿，包含正文抽取、图片 OCR 与人工复核提示

Notes:

- 本目录是上游原始材料区，默认只读。
- 这里的内容不能直接视为“当前代码已实现能力”。
- 同目录下的 `*.pdf` 为通过本机 Word 高保真导出的阅读副本，不替代原始 `docx`。
