## Rule

If product scope changes, update this directory first, then update `Agent.md`, `docs/engineering/`, and `docs/project_management/task_board.md` as needed.

Current rule of thumb:

- `00` to `07` define the current MVP truth.
- `08` and `10` may describe future or brand-layer expansion, and must not override current MVP boundary.
