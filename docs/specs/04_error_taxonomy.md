这是错因编码的唯一权威来源。

## 编码体系

错因编码分两类：

- **知识类 (E-K)**: E-K01 ~ E-K23 — 知识理解不足导致的错误
- **习惯类 (E-H)**: E-H01 ~ E-H21 — 习惯性粗心导致的错误
- **未知**: E99 — 暂未识别的错因

详细错因类型文件见 `knowledge_base/01_error_taxonomy/`（44 个 Markdown 文件）。

## 知识类错因 (E-K)

| Code | Label | Typical Signal | Teacher Action |
| --- | --- | --- | --- |
| `E-K01` | 基础事实错误 | 口诀、口算、基础加减乘除事实错误 | 短时低负荷练习，要求说出计算依据 |
| `E-K02` | 进位错误 | 加法或乘法中满十进位遗漏或未加 | 显式标注进位，再做同结构少量练习 |
| `E-K03` | 退位错误 | 减法本位不够减时退位处理错误 | 用数位表复盘退位过程 |
| `E-K04` | 数位对齐错误 | 竖式错位、小数位或部分积位置错误 | 用方格纸/数位表重写 |
| `E-K05` | 运算顺序错误 | 混合运算未按括号或先乘除后加减 | 先圈出第一步再计算 |
| `E-K06` | 小数点/分数单位错误 | 小数点位置、分子分母或分数单位错误 | 先估算范围或统一分数单位 |
| `E-K07` | 抄题/转写错误 | 数字、符号、条件从题目到步骤变化 | 圈画原题数字和符号后再列式 |
| `E-K08` | 步骤遗漏 | 漏部分积、漏商位、跳过关键中间步骤 | 要求补齐每步 |
| `E-K09` | 算理理解不足 | 方法选择错误或迁移失败 | 用图示、操作、反例讲清算理 |
| `E-K10` | 审题与单位理解错误 | 问题对象或单位理解错误 | 复述问题，标注单位 |
| `E-K11` | 分数运算错误 | 分数加减未通分、乘除未约分 | 用图形辅助理解分数运算 |
| `E-K12` | 比例/百分数错误 | 比例关系混淆、百分数转化错误 | 用具体例子讲清比例含义 |
| `E-K13` | 面积/体积公式错误 | 公式记忆错误或单位换算错误 | 用操作活动理解公式推导 |
| `E-K14` | 方程列式错误 | 等量关系找错、方程列错 | 用天平模型理解等式 |
| `E-K15` | 图形认识错误 | 图形特征识别错误 | 用实物操作加深理解 |
| `E-K16` | 统计图表错误 | 数据读取或图表理解错误 | 用实际数据分析练习 |
| `E-K17` | 单位换算错误 | 进率记错、换算方向错误 | 用数轴或具体参照物 |
| `E-K18` | 正负数运算错误 | 符号处理错误 | 用数轴理解正负数 |
| `E-K19` | 圆的计算错误 | 圆周率、面积公式使用错误 | 用圆形纸片操作 |
| `E-K20` | 比例尺错误 | 比例尺计算、单位换算错误 | 用地图实例练习 |
| `E-K21` | 统计量错误 | 平均数、中位数计算错误 | 用具体数据集练习 |
| `E-K22` | 可能性判断错误 | 概率概念混淆 | 用实验验证 |
| `E-K23` | 综合应用错误 | 多步骤问题中策略选择错误 | 分解步骤逐个击破 |

## 习惯类错因 (E-H)

| Code | Label | Typical Signal | Teacher Action |
| --- | --- | --- | --- |
| `E-H01` | 未验算 | 结果明显不合理但未发现 | 引入估算、逆运算、代入检查 |
| `E-H02` | 抄写错误 | 答案正确但抄到答题处时出错 | 养成对照原题检查习惯 |
| `E-H03` | 漏题 | 跳过未做 | 做完逐题核对 |
| `E-H04` | 计算粗心 | 简单计算出错 | 放慢速度，逐步计算 |
| `E-H05` | 审题不仔细 | 漏看条件或单位 | 圈画关键词 |
| `E-H06` | 书写潦草 | 数字看错导致计算错误 | 规范书写 |
| `E-H07` | 过度自信 | 不检查直接交 | 建立检查习惯 |
| `E-H08` | 紧张出错 | 考试时会做但做错 | 模拟练习减压 |
| `E-H09` | 依赖计算器 | 过度依赖工具 | 限时口算练习 |
| `E-H10` | 跳步计算 | 省略中间步骤导致错误 | 要求写出完整步骤 |
| `E-H11` | 书写格式不规范 | 竖式不整齐、数位不对齐 | 用方格纸规范格式 |
| `E-H12` | 不画图辅助 | 几何题不画图导致错误 | 培养画图习惯 |
| `E-H13` | 不读题直接做 | 没理解题意就下笔 | 先复述题意再动笔 |
| `E-H14` | 检查时只看不算 | 检查流于形式 | 教检查方法（代入、估算） |
| `E-H15` | 时间分配不当 | 前面太慢后面匆忙 | 练习时间管理 |
| `E-H16` | 不整理错题 | 同类错误反复出现 | 建立错题本 |
| `E-H17` | 不及时订正 | 错了不改 | 当天订正制度 |
| `E-H18` | 做题顺序不当 | 先难后易导致心态崩 | 建议先易后难 |
| `E-H19` | 不标记疑问 | 有不懂的不问 | 鼓励提问 |
| `E-H20` | 答题不完整 | 漏写单位、答句 | 检查答题完整性 |
| `E-H21` | 不按要求作答 | 没看清答题要求 | 审题后标注要求 |

## MVP 覆盖范围

### 规则引擎已覆盖

- E-K01 ~ E-K08（基础事实、进位、退位、数位对齐、运算顺序、小数点、抄题、步骤遗漏）
- E-K11（分数运算）
- E-H01（未验算）

### 部分覆盖

- E-K06（小数点/分数单位）
- E-K09（算理理解）
- E-K10（审题与单位）

### 未覆盖

- E-K12 ~ E-K23（比例、面积、方程、图形、统计等）
- E-H02 ~ E-H21（大部分习惯类）

## 诊断输出规则

每次诊断必须包含：

- Code（错因编码）
- Label（错因标签）
- Confidence（置信度 0-1）
- Evidence（诊断证据）
- Teacher action（教师处理建议）
- Student-facing hint（学生可读提示）
- Review status（审核状态）
