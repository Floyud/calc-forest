## 二、成长里程碑（一棵树的9阶段）

| 阶段 | 累计天数 | 变化 | 寓意 |
| --- | --- | --- | --- |
| 播种 🌱 | 第1天 | 种子入土 | 开始 |
| 破土 | 第3天 | 小芽破土 | 坚持3天就有变化 |
| 第一片叶 | 第7天 | 长出真叶 | 一周看到成果 |
| 长高 | 第14天 | 明显长高 | 两周成长可见 |
| 分支 | 第21天 | 长出分支 | 三周开始成形 |
| 茁壮 | 第30天 | 小树茁壮 | 一月小有成就 |
| 花苞 | 第45天 | 出现花苞 | 一个半月期待 |
| 开花 🌸 | 第60天 | 开花 | 两月美丽绽放 |
| 成材 🌳 | 第90天 | 长成大树 | 三月见证坚持 |

### 数据模型

```json
{
  "growth_milestone": {
    "current_stage": "branching",
    "days_completed": 21,
    "total_days_in_cycle": 90,
    "cycle_type": "semester",
    "tree_species": "cherry"
  }
}
```

里程碑判定规则：
- 纯基于累计练习天数，不依赖连续打卡
- 一个成长周期 = 一个学期或一个假期
- 达标条件：学期/假期内练习天数 ≥ 周期天数的 50%

## 三、树种系统（历史规划：8种核心 + 后续扩展）

### 首批树种设想（历史规划）

| 类别 | 树种 | 教育价值 | 限定周期 |
| --- | --- | --- | --- |
| 果树 | 🍎 苹果树 | 结果快，建立成就感 | 学期 |
| 果树 | 🍊 橘子树 | 结果快，建立成就感 | 学期 |
| 观赏 | 🌸 樱花树 | 培养审美 | 学期 |
| 观赏 | 🍁 枫树 | 培养审美 | 学期 |
| 常绿 | 🌲 松树 | 理解坚韧 | 学期 |
| 常绿 | 🌳 橡树 | 树立远大理想 | 学期 |
| 寒假限定 | ❄️ 腊梅 | 凌寒独开 | 仅寒假 |
| 暑假限定 | 🌻 向日葵 | 向阳生长 | 仅暑假 |

树种不是单纯的界面皮肤。它们承担三层价值：

- 给孩子“这是我的树”的拥有感。
- 让数学练习顺带接触自然科学、季节观察和文化寓意。
- 把长期坚持表达为可感知、可讲述、但不带比较压力的成长过程。

## 四、森林积累体系

### 按起始年级的森林展望

| 起始年级 | 可种树数 | 森林形态 | 寄语 |
| --- | --- | --- | --- |
| 一年级 | 24棵 | 茂密大森林 | "六年，你拥有了一片大森林！" |
| 二年级 | 20棵 | 美丽森林 | "五年，你的森林真茂盛！" |
| 三年级 | 16棵 | 舒适森林 | "四年，你的森林很舒适！" |
| 四年级 | 12棵 | 小森林 | "三年，你拥有了一片小森林！" |
| 五年级 | 8棵 | 小树林 | "两年，你的小树林很珍贵！" |
| 六年级 | 4棵 | 精致树林 | "一年，你种下了属于自己的成长礼物！" |

**核心原则：** 森林没有"标准大小"。你的森林，就是你坚持的见证。

## 五、四季成长周期

| 周期 | 时长 | 产出 | 限定树种 |
| --- | --- | --- | --- |
| 上学期 | 约4个月 | 一棵树 | 常规树种 |
| 寒假 | 约1个月 | 一棵冬日小树 | ❄️ 腊梅、竹子 |
| 下学期 | 约4个月 | 一棵树 | 常规树种 |
| 暑假 | 约2个月 | 一棵夏日小树 | 🌻 向日葵、荷花 |

### 假期模式原则

- **定位：** 复习巩固、温故知新，不提前学
- **题目来源：** 100%本学期已学内容
- **时长：** 每日≤10分钟
- **参与方式：** 完全自愿，不推送通知
- **打卡：** 不计算连续打卡
- **排名：** 不做班级统计
- **达标条件：** 假期累计练习天数 ≥ 假期天数的一半

## 六、情感化鼓励体系

### 里程碑鼓励

| 场景 | 鼓励语 |
| --- | --- |
| 完成练习 | "今天小树苗喝饱了水，长高了一点点！" |
| 累计3天练习 | "累计练习 3 天！小树苗破土而出了！" |
| 累计7天练习 | "累计练习 7 天！小树苗长出了新叶子！" |
| 累计30天练习 | "累计练习 30 天！小树长得更茁壮了！" |
| 自己发现错误 | "太棒了！你自己发现了问题所在！这就是思考的力量~" |

### 按年级鼓励

| 起始年级 | 鼓励语 |
| --- | --- |
| 一年级 | "你有六年时间，可以种出一片大大的森林！" |
| 三年级 | "你还有四年，可以种出16棵树，从现在开始正好！" |
| 五年级 | "你还有两年，可以种出8棵树，每一棵都很珍贵！" |
| 六年级 | "你有一年时间，可以种出4棵树。即使只有4棵，也是属于你的成长森林！" |

## 八、与现有架构的关系

```
现有 MVP 流程（保留）:
  合成作答记录 → FastAPI 诊断 API → 错因标签 + 证据 → 教师审核

新增森林层（叠加，不替代）:
  诊断结果 → 练习完成 → 更新成长天数 → 推进里程碑 → 种树成长
                                    ↓
                              森林视图 / 鼓励语

教师仍审核一切。森林成长是练习数据的可视化呈现，不影响诊断准确性。
```
